Feature Roadmap

Vib.Timer is a WatchOS app that sends subtle vibration reminders to your wrist.

Whether you're concentrating on work, practicing meditation, engaging in a workout, or handling everyday tasks, Vib.Timer helps you stay aware of the passing time without distractions. All features are fully accessible to every user — no subscriptions, no ads.

Download Vib.Timer from the App Store today, completely free!

Below is an overview of the features planned for future releases:

Recently Added

✓ Start a 5-minute timer with vibrations every 30 seconds
✓ One-Tap Control: Start, pause, or reset the timer with a single tap
✓ Real-Time Display: See the elapsed time counting up

Upcoming Features

☆ Set custom timer and interval settings
☆ Customize vibration patterns & strengths
☆ End of Session: Display total elapsed time and the number of interval loops
☆ Session History: Keep a record of previous timer sessions
☆ Saved presets for easy access
☆ Countdown Mode: Set a countdown timer and get alerts as it counts down
☆ Experiment with UI design and color variations
☆ Simple Streaks: Basic one session a day
